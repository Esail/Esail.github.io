configuration

通用

parameter lightgbm param default type description
prePartition is_pre_partition True bool true 如果训练数据 pre-partitioned, 不同的机器使用不同的分区
boostingType boosting_type gbdt string gbdt,rf, dart, goss
parallelism tree_learner data_parallel string 数据并行的 tree learner
defaultListenPort 无 12400 int The default listen port on executors, used for testing
numIterations num_iterations 100 int boosting 的迭代次数
learningRate learning_rate 0.1 double 学习率
numLeaves num_leaves 31 int 一棵树上的叶子数
maxBin max_bin 255 int 工具箱的最大数特征值决定了容量 工具箱的最小数特征值可能会降低训练的准确性, 但是可能会增加一些一般的影响（处理过度学习）LightGBM 将根据 max_bin 自动压缩内存。 例如, 如果 maxbin=255, 那么 LightGBM 将使用 uint8t 的特性值
baggingFraction bagging_fraction 1.0 double 类似于 feature_fraction, 但是它将在不进行重采样的情况下随机选择部分数据 可以用来加速训练 可以用来处理过拟合 Note: 为了启用 bagging, bagging_freq 应该设置为非零值
baggingFreq bagging_freq 0 int bagging 的频率, 0 意味着禁用 bagging. k 意味着每 k 次迭代执行bagging Note: 为了启用 bagging, bagging_fraction 设置适当
baggingSeed bagging_seed 3 int Bagging seed
earlyStoppingRound early_stopping_round 0 int Early stopping round
featureFraction feature_fraction 1.0 double Feature fraction
maxDepth max_depth -1 int -1意味着没有限制
minSumHessianInLeaf min_sum_hessian_in_leaf 1e-3 Double Minimal sum hessian in one leaf
无 num_machines 1 int 并行运行的机器数量，在spark上，有多少个executor就有多少个机器
timeout 无 120s second socket超时
objective application regression enum regression, regression_l1, huber, fair, poisson, quantile, quantile_l2, binary, multiclass, multiclassova, xentropy, xentlambda, lambdarank
min_data_in_leaf min_data_in_leaf 20 int min_data_in_leaf
min_data_in_bin min_data_in_bin 3 int min_data_in_bin
lambda_l1 lambda_l1 0 double lambda_l1
lambda_l2 lambda_l2 0 double lambda_l2
modelString 无 “” String model存成String格式

##分类

parameter lightgbm param default type description
无 metric binary_logloss,auc String 度量

##回归

parameter lightgbm param default type description
alpha alpha 0.9 double Huber loss 和 Quantile regression 的参数. 将用于 regression 任务
tweedieVariancePower tweedie_variance_power 1.5 double control the variance of tweedie distribution, must be between 1 and 2
